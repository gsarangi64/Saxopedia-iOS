# Saxopedia

## Description:
This app is a comprehensive saxophone repertoire manager that consumes two web APIs to provide musicians with detailed information about classical saxophone pieces and their composers. The app features dedicated API views displaying real-time data from the Open Opus API and a custom saxophone repertoire JSON API, with full localization support for English, Spanish, and French. Users can browse an extensive collection of saxophone pieces, view detailed composer information, manage application settings with persistent storage, and receive practice reminders through local notifications.

## Features
- Dual API Integration: Real-time composer data from Open Opus API and saxophone repertoire from custom JSON API
- Practice Reminder System: Daily notifications with customizable timing
- Full Localization: Complete Spanish and French translations using String Catalogs
- Animated UI: Smooth transitions and loading animations throughout the app
- Persistent Settings: 4+ user preferences saved using @AppStorage
- Custom Color Scheme: Saxophone-themed gold and cream colors
- Launch Screen: Custom designed launch screen with saxophone imagery

## APIs Used:
- Open Opus API (https://api.openopus.org) - For composer information
- Saxophone Repertoire JSON API (https://raw.githubusercontent.com/gsarangi64/sax-repertoire-data/main/sax_repertoire.json) For piece details
